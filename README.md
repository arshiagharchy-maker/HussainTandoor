# Hussain Tandoor Website

## Overview
A modern, responsive website for Hussain Tandoor, featuring a product catalog, team section, and contact/social links. The site is built with HTML and CSS, and includes individual product pages for up to 20 products.

## Structure
```
hussain-tandoor-website
├── src
│   ├── index.html                # Main landing page
│   ├── css/
│   │   ├── base.css
│   │   ├── styles.css
│   │   ├── layout.css
│   │   └── components.css
│   ├── products/
│   │   ├── product-1.html        # Product detail pages (1-20)
│   │   ├── product-2.html
│   │   └── ... up to product-20.html
│   ├── images/                   # Images for hero, products, etc.
│   └── support.html              # Support/contact page (to be added)
├── .gitignore
└── README.md
```

## Features
- Hero section with images and call-to-action
- Product catalog with individual product pages
- Team section with contact info and WhatsApp links
- Footer with copyright, contact email/phone, and social links

## Getting Started
1. Clone this repository.
2. Open `src/index.html` in your browser.
3. To view product details, click "Learn More" on any product card.

## Customization
- Update product pages in `src/products/` with real product info and images.
- Add or update team member info in the team section.
- Edit footer links for support, privacy, or other pages as needed.
